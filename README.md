# 义乌市洪瑞劳务服务有限公司 - 官方网站

![洪瑞劳务 Logo](./assets/images/logo.svg)

## 📖 项目简介 | Introduction

本项目为 **义乌市洪瑞劳务服务有限公司** 的官方企业落地页。网站旨在展示公司在人力资源领域的专业服务，包括劳务派遣、人事外包、人才招聘及职业培训等核心业务，并提供便捷的业务联系方式。

## ✨ 特性 | Features

- **高端视觉设计**：采用“企业蓝”为主色调，融合磨砂玻璃（Glassmorphism）与渐变效果，呈现专业、现代的企业形象。
- **响应式布局**：完美适配 PC、平板及手机端，提供流畅的跨设备浏览体验。
- **动态交互**：
  - AOS 滚动视差动画，提升页面生动感。
  - 导航栏及按钮悬停特效，增强用户操作反馈。
  - 移动端折叠菜单，优化小屏操作体验。
- **贴心功能**：
  - **一键拨号**：点击电话号码即可自动跳转拨号。
  - **业务直连**：显眼的微信二维码及联系方式，方便客户快速对接。

## 🛠️ 技术栈 | Tech Stack

- **HTML5**: 语义化结构。
- **Tailwind CSS** (CDN): 实用主义 CSS 框架，实现快速样式定制。
- **Vanilla JavaScript**: 轻量级原生 JS，负责交互逻辑。
- **AOS.js**: 页面滚动动画库。
- **FontAwesome**: 矢量图标库。

## 🚀 快速开始 | Quick Start

### 本地运行

本项目为纯静态网站，无需复杂的构建环境。

1. 克隆仓库：

   ```bash
   git clone https://github.com/AIGC-OFFICE/HRs.git
   ```

2. 直接双击打开 `index.html` 即可在浏览器中预览。

### 部署建议

推荐使用 **Netlify** 或 **Vercel** 进行免费托管：

- 将项目文件夹直接拖入 Netlify Drop 区域即可上线。
- 或连接 GitHub 仓库实现自动部署。
