## 2. 合规红线（法律顾问把关 · 硬约束，所有页面必须遵守）

1. **禁止点名竞品**：不得出现「自如」「苏宁」等任何竞品名。对比一律用中性词「互联网标准化平台 / 流量型平台」。（《反不正当竞争法》第11条商业诋毁 / 《广告法》）
2. **除甲醛禁绝对化 / 疗效用语**：禁止「保证 / 100% / 彻底清除 / 根除 / 无害 / 治愈过敏」等。只描述工艺、检测、复检（如「熏蒸解离+多重喷涂闭环工艺，配备气相色谱等专业仪器多点检测，两次以上上门复检」）。（《广告法》第4/28条 + 绝对化用语禁令）
3. **禁医疗 / 健康功效承诺**：除螨 / 除菌作为「处理工艺」可写，不得承诺「防过敏 / 治疗 / 健康」结果。
4. **删除未经核实的行业数字**：不得出现「200%年增速」「60%收纳师年薪破10万」「万元级物损」等文档内部数据；改为定性表述（如「收纳需求增长迅速」）。
5. **禁内部战略黑话**：不得出现「特洛伊木马 / 筛选低价格敏感客户 / 对冲物损风险 / 防御性消费」等。一律翻译成客户视角利益（如「为进口大理石、艺术涂料、定制实木提供防护性清洁，避免清洁不当造成的损伤」）。
6. **价格真实性**：所有价格视为真实对外报价，保留「起 + 上门核定」。
7. **设备/承诺真实性**：「气相色谱仪」「验收满意再付款」属真实承诺，文案保留但须企业实际可兑现（已在交付说明中提示用户）。
